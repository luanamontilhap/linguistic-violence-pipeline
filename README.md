# Linguistic Violence Pipeline

Implementação do Stage 3 (Ontological Mapping) do artigo 
"Modeling Linguistic Violence: An Ontology-Based Framework 
for the Computational Analysis of Violence Manifested in Language"

Classifica textos em categorias de violência linguística:
- Hate Speech
- Dangerous Speech  
- Verbal Aggression
- Toxic Speech
- Cyberbullying
- Benign

## Como usar
python stage3.py
